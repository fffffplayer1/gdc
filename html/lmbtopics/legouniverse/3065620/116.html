JAMMERS ON! RANGERS ONLY! NO SPREADING OF INFO! NO ONE IN OR OOC WHO IS NOT AUTHORIZED!

The Hackhelmet is complete!

Okay, here's the facts. It's a pilot's Helmet, except lighter, and with a different heads up display. The display links into a remote computer, so there's no actual processing in the helmet so as to save weight. This also means that there is nothing that can be reverseengineered out of the helmet, except of course the transmitter.

Therefore, the helmet has a special mic system for analyzation of the voice of the user. So each helmet has to be calibrated to the user. Anyway, the helmet links through a transmitter to a remote computer system which manages and stores data and such for the helmets. The link is done through a special electromagnetic-infrared system. So there are some limits for connectivity.

Now for the features. In addition to being light and all the other cool stuff, like being able to remotely control and hack and stuff the helmet also has Near-Field Communication and Wi-fi. The NFC allows for helmets to share info wirelessly when within a close area, so that if jamming is occurring, the users can still transmit data to each other. The Wi-Fi is not for communication, but rather, it allows for users of it to see behind walls and such. The signals can illuminate objects on the other side. I've also built in some of the standard sensors. Infra-red, Imagination, etc.

Thoughts?
