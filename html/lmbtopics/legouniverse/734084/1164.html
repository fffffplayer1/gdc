My desgn for such a system would be as follows. We could desgn a large, sealed container/chamber that is connected to individual ventilation shafts that run throughout the fortress. If a vacuum power was used inside this sealed contaner/chamber, air would rush into it via the only access points, the ventilation shafts. Since these shafts open up in different rooms throughout the fortress, they would be taking air from those rooms and sucking them through and into this sealed chamber which would fill up with the air collected from the outside rooms. We would have a system that would shut off the ventilation shafts once the chamber was filled thus preventing any air from returning to the outside rooms. That would be the basic principle. We could expand upon it by building more then one sealed chamber and connecting them to different portions of the fortress. All we would really need to figure out is how to accomplish the release of a vacuum power inside these chambers. Does this make any sense? This actually would be an effective defense system as we could simply drain the air out of select rooms instead of everything at once.
